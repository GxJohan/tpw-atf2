Instrucciones:

A continuación, se presenta un código HTML básico sobre una tienda de plantas. Tu tarea es mejorar y personalizar este sitio web utilizando CSS y siguiendo las indicaciones proporcionadas. Asegúrate de cumplir con todos los requisitos para obtener la puntuación completa.

Código HTML base:


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Jardín Botánico Online</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Jardín Botánico Online</h1>
        <nav>
            <ul>
                <li><a href="#inicio">Inicio</a></li>
                <li><a href="#productos">Productos</a></li>
                <li><a href="#sobre-nosotros">Sobre Nosotros</a></li>
                <li><a href="#contacto">Contacto</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="inicio">
            <h2>Bienvenidos a nuestro jardín virtual</h2>
            <p>Descubre la belleza de la naturaleza desde la comodidad de tu hogar.</p>
        </section>

        <section id="productos">
            <h2>Nuestras Plantas</h2>
            <div class="galeria-productos">
                <div class="producto">
                    <img src="https://via.placeholder.com/200" alt="Planta 1">
                    <h3>Planta 1</h3>
                    <p>Descripción de la planta 1.</p>
                    <button>Comprar</button>
                </div>
                <!-- Repite este bloque para las plantas 2, 3 y 4 -->
            </div>
        </section>

        <section id="sobre-nosotros">
            <h2>Sobre Nosotros</h2>
            <p>Somos una tienda en línea dedicada a llevar la naturaleza a tu hogar.</p>
        </section>

        <section id="contacto">
            <h2>Contacto</h2>
            <form>
                <label for="nombre">Nombre:</label>
                <input type="text" id="nombre" name="nombre" required>

                <label for="email">Email:</label>
                <input type="email" id="email" name="email" required>

                <label for="mensaje">Mensaje:</label>
                <textarea id="mensaje" name="mensaje" required></textarea>

                <button type="submit">Enviar</button>
            </form>
        </section>
    </main>

    <footer>
        <p>&copy; 2024 Jardín Botánico Online. Todos los derechos reservados.</p>
    </footer>
</body>
</html>

Tareas y puntuación:

1. Personalización con CSS y tipografía (4 puntos)

  • Utiliza una fuente personalizada de Google Fonts para los títulos. (1 punto) Sugerencia: Explora fuentes como 'Playfair Display' o 'Montserrat' para los títulos.
  • Aplica una fuente diferente para el texto del cuerpo. (1 punto) Considera usar 'Roboto' o 'Open Sans' para el texto principal.
  • Crea un esquema de colores coherente para todo el sitio. (1 punto) Utiliza tonos verdes y marrones para reflejar la naturaleza.
  • Implementa un diseño de encabezado atractivo con un logotipo (puede ser texto estilizado). (1 punto) Puedes usar CSS para crear un logotipo simple con texto y un ícono de hoja.

2. Iconos y elementos visuales (3 puntos)

  • Agrega íconos de Font Awesome a los elementos de navegación. (1 punto) Usa íconos como 'fa-home', 'fa-leaf', 'fa-info-circle' y 'fa-envelope'.
  • Incluye un ícono decorativo en la sección "Sobre Nosotros". (1 punto) Considera un ícono grande de planta o árbol como fondo.
  • Crea un botón de "scroll to top" con un ícono de flecha. (1 punto) Usa 'fa-arrow-up' y posiciónalo fijo en la esquina inferior derecha.

3. Animaciones y transiciones (4 puntos)

  • Crea una transición hover para los botones de "Comprar". (2 puntos) Usa 'transform: scale()' y cambia el color de fondo al hacer hover.
  • Implementa una animación de carga (spinner) que se muestre brevemente al cargar la página. (1 punto) Crea un div con 'border-radius: 50%' y anímalo con '@keyframes'.
  • Añade una transformación a las imágenes de los productos al hacer hover. (1 punto) Aplica un ligero zoom o rotación a las imágenes.

4. Diseño responsivo (4 puntos)

  • Asegúrate de que el diseño sea completamente responsivo y se adapte a dispositivos móviles, tablets y desktops. (2 puntos) Usa 'max-width' y porcentajes para elementos flexibles.
  • Implementa un menú hamburguesa para dispositivos móviles. (1 punto) Crea un botón con tres líneas horizontales que muestre/oculte el menú.
  • Utiliza al menos una media query para ajustar el diseño en diferentes tamaños de pantalla. (1 punto) Ajusta el layout a una columna para móviles (max-width: 768px).

5. Flexbox y Grid Layout (5 puntos)

  • Utiliza Flexbox para la navegación del encabezado. (1 punto) Usa 'display: flex' y 'justify-content: space-between'.
  • Implementa Grid Layout para la galería de productos, asegurando que se ajuste responsivamente. (2 puntos) Usa 'display: grid' y 'grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))'.
  • Usa Flexbox o Grid para estructurar el formulario de contacto. (1 punto) Organiza los campos del formulario en dos columnas con Flexbox.
  • Aplica Flexbox o Grid para organizar el contenido en la sección "Sobre Nosotros". (1 punto) Usa Flexbox para alinear el texto y la imagen decorativa.

  • Incluye una breve explicación de las decisiones de diseño que tomaste.
